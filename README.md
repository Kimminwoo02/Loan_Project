# Loan_Project
대출 상담, 신청, 이용약관, 심사 정보 등 대출에 관련된 프로세스를 학습하고 비즈니스 로직에 관하여 공부하기 위해 작업한 프로젝트

## 개발 환경

* Intellij IDEA Ultimate
* Java 11
* Gradle 
* Spring Boot 2.7.1

## 기술 세부 스택

* Spring Web
* Spring Data JPA
* MySQL Driver
* Lombok
* Spring Boot DevTools
* Spring Configuration Processor

그 외

* QueryDSL
